# artinscience · scholarly

The **manuscript / engraving tradition** — botanical, anatomical, geometric,
astronomical. The story of *knowledge*, hatch-shaded and Latin-labeled, drawn as
pen-plotter line work.

**Series within this line:**
- **Byrne's Euclid, extended** — recreate/continue Oliver Byrne's colored,
  diagram-only *Elements*; geometric proof as art.
- **Botanical / herbarium plates** — plants + uses; Latin binomial ↔ folk name;
  engraving hatching.
- **Anatomical plates** — Vesalius-style; body systems, Latin ↔ English.
- **Patent drawings** — historical inventions + the patent text (public domain).
- **Star atlases & constellations** — real catalogs (Gaia/Hipparcos) as a
  celestial atlas; cross-cultural star myths.
- **Periodic table / elements** — real emission **spectra** plotted as the art.
